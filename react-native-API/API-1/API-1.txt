1. splash
1-1. SplashScreen.hideAsync(); => Splash 화면을 즉시 숨긴다.
1-2. SplashScreen.preventAutoHideAsync(); => Splash 화면을 즉시 나타낸다.

app.json의 splash 설정을 통해 이미지와 background 처리를 할 수 있다.


2. flatList
2-1. data 속성
=> rendering 할 떄 필요한 data

2-2. keyExtractor
=> key props을 설정할 떄 사용하는 속성
=> 함수를 준다.
=> 함수를 줄 떄 매개변수에 자동적으로 item과 index가 포함된다. item의 id, key을 return 하면 된다.


2-3. renderItem
=> 렌더링할 컴포넌트 + 데이터를 규정한다.
=> 함수를 주면 된다. 
=> 매개변수에 item = {item : {data....}} 이런 식으로 담겨있다.
=> 두 번 풀어서 사용해야 데이터가 나온다.
